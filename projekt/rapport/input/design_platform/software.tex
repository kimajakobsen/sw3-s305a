\section{Software}
\label{sec:software}

Our system relies on two things:

\begin{itemize}
	\item Database
	\item Web server
\end{itemize}

\subsection{DBMS}
\label{subsec:dbms}

Our system requires a DBMS supporting the query language \sql[] and the relational database storage model.
We have chosen Microsoft SQL server in favor of \posgresql[], wich we have past experience with, but is less integrated with Visual Studio 2010, which we are using for developing our system.\fixme{Inds\ae t Microsoft SQL server versionsnummer}
